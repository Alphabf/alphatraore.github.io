Independent QC Engine (ADaM/TFL validation checks)

Independent QC framework to validate ADaM datasets and TFL outputs using automated checks, summaries, and reconciliation-ready reporting.

What it does

  • Record counts & key-level uniqueness checks
  • Derivation checks (baseline, flags, visits, parameters)
  • Consistency checks across datasets (subject/visit/param)
  • Output reconciliation summaries + log review helpers

Tools: SAS / R, validation reports
